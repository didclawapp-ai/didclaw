DidClaw · Logo 概念稿

D 字母 × 龙虾拟态 — 5 个方向参考

方案 A
轮廓剪影
D 笔画即身体，竖笔=腹甲，弧笔=背甲
大螯从上下探出，橙红渐变
方案 B
几何扁平
低多边形拼色，D 轮廓清晰
适合 App 图标 / 小尺寸
方案 C
线描徽章
圆形徽章底盘 + 单色线描
适合水印、印章、深色背景
方案 D
霓虹赛博
暗底 + 橙红霓虹发光
适合深色主题 / 启动页
方案 E
极简方块 / App 图标
圆角方形底盘，负空间白色 D
最适合 Tauri App 图标 / 安装包
设计说明
A — 轮廓剪影
最还原「龙虾感」，D 字笔画即壳体，适合横版 Logo 或大图
B — 几何扁平
低多边形风格，色块清晰，小尺寸仍可辨认
C — 线描徽章
圆形徽章，单色线描，适合文档水印 / 标章
D — 霓虹赛博
暗底橙红发光，与 DidClaw 暗色 UI 风格匹配
E — App 图标
白色负空间 D，圆角方块底盘，最适合 Tauri 图标文件
