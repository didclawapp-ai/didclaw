实时 Gateway 连接

WebSocket 与 OpenClaw Gateway 保持长连接，状态灯实时显示，一键连接/断开，消息即达秒响应。

Skills 技能生态

ClawHub 在线市场 + OpenClaw 工作区一体管理。支持搜索安装、启用/禁用、从本地文件夹或 ZIP 安装自定义 Skills。

定时任务调度

内置 Cron 任务管理器，支持按时间点 (at)、间隔 (every)、Cron 表达式三种方式调度，查看执行历史与运行记录。

多会话并发

同时管理多个 Agent 会话，后台会话有活动时标签闪动提醒。完整会话历史，Token 用量实时统计。

本地优先，隐私安全
