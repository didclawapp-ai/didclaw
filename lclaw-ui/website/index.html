核心功能

一款应用，全面掌控你的 AI

从多渠道消息接入到本地 Skills 生态，DidClaw 让 AI 真正为你所用。

多渠道消息接入

支持微信、WhatsApp、Slack、Discord、LINE、飞书等主流渠道，在你熟悉的 App 里直接指挥 AI，无需切换工具。

💬 微信 📱 WhatsApp 🔷 Slack 🎮 Discord ✈️ LINE 🪶 飞书

实时网关连接

通过 WebSocket 与 OpenClaw Gateway 保持长连接，消息即达，任务秒响应。支持一键断连与重连。

Skills 技能生态

丰富的 Skills 插件市场，涵盖文件操作、网页浏览、代码执行等。支持自定义 Skills，无限扩展 AI 能力边界。

本地优先，隐私安全

所有会话数据存储在本机，无需担心数据上传云端。支持完整备份与恢复，你的数据完全由你掌控。

会话管理

历史会话随时查阅，支持新建、切换多会话。每个 Agent 拥有独立上下文，精准执行任务互不干扰。

多模型支持

内置 MiniMax、OpenAI、Google 等主流 AI 模型接入，支持 API Key 与 OAuth 两种鉴权方式，一套客户端，随时切换模型。

🤖 MiniMax ✨ OpenAI 🔵 Google Gemini 🟤 Ollama
